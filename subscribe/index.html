Common Questions
What is an API key and why do I need one?
Your API key is a signed JWT (JSON Web Token) that authenticates your tier level. Use it as a Bearer token in any request to the CanonMind API or signal-mcp server. It grants access to the 60-entry sealed Canon KB according to your subscription tier.
What are Rule 06 works?
Four works in the KB are classified as Rule 06 Specialist: Andrei Rublev, The Mirror, Stalker, and Au Hasard Balthazar. These require prerequisite work completion and are only available on Canon and Institution tiers.
What is signal-mcp?
signal-mcp is a Model Context Protocol server (github.com/Rationaloptimist140/Canonmind.com/tree/main/mcp/signal-mcp) that lets LLM agents and developer tools query the sealed 60-entry Canon KB. Install via npm and configure in Claude Desktop or Cursor.
How does the 60-entry cap work?
The Canon KB is permanently sealed at 60 entries under Constant 08. New works can only enter by displacing an existing entry through formal CanonGuardian batch review. The cap is a design decision, not a technical limitation — it is the platform's core quality guarantee.
